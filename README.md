# 企业ERP管理系统

一个功能完整的企业资源管理系统，适合中小企业使用。

**货币单位：马来西亚令吉（RM）**

## 核心功能

### 1. 基础数据管理
- 商品/物品管理（支持1000+种物品）
- 客户管理
- 供应商管理
- 员工管理

### 2. 仓库管理
- 实时库存查询
- 进货管理
- 出货管理
- 库存调整
- 库存预警
- 进出库历史记录

### 3. 活动管理
- 促销活动管理
- 工作坊项目管理
- 展会活动管理
- 自定义活动类型
- 活动物料分配

### 4. 订单管理
- 销售订单
- 采购订单
- 发票管理

### 5. 财务管理
- 收支记录
- 财务报表
- 按活动核算利润

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- IndexedDB (本地数据库)
- React Router

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 数据备份

系统使用IndexedDB本地存储，建议定期导出数据备份。

## 使用说明

1. 首次使用建议先添加商品信息
2. 设置库存预警值
3. 创建活动后可以关联物料出库
4. 所有操作都会自动记录历史

## 支持的浏览器

- Chrome (推荐)
- Edge
- Firefox
- Safari

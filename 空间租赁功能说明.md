# 📍 空间租赁管理功能说明

## 功能概述

为半亩天光智慧中心量身定制的空间租赁管理系统，支持多个空间的管理、预订和日历可视化。

---

## 🏢 可租赁空间

根据您的需求，系统支持管理以下空间：

1. **半亩天光讲堂**
2. **半亩天光阅读区**
3. **半亩天光**
4. **T&N**

---

## 🎯 核心功能

### 1. 空间管理
- 添加/编辑/删除空间
- 设置租金（时租/日租/月租）
- 管理空间设施
- 空间状态管理

### 2. 活动关联
- 活动管理中直接选择空间
- 自动关联场地信息
- 空间使用情况追踪

### 3. 日历视图
- 可视化展示所有业务活动
- 显示收入、场地、活动
- 月度视图快速查看

## 📊 日历视图功能详解

### 颜色标识

**日期背景：**
- 🔵 蓝色边框：今天
- ⚪ 白色背景：普通日期
- 🟦 浅蓝背景：今天的日期

**事件标签：**
- 🟣 紫色：活动（📅 X活动）
- 🟢 绿色：空间预订（📍 X预订）
- 🟡 黄色：收入（💰 RMXXX）

### 信息聚合

**每个日期显示：**
1. 活动数量 - 当天举办的活动
2. 预订数量 - 当天的空间预订
3. 收入金额 - 当天所有收入（销售+租金）

**点击日期显示：**
- 活动列表（名称、地点）
- 预订列表（空间、客户、金额）
- 销售笔数和金额
- 当日总收入

## 🎯 使用建议

### 1. 空间定价策略

**时租 vs 日租 vs 月租：**

```
建议定价：
- 时租: 适合短期会议、小型活动
- 日租: 适合全天活动、讲座
- 月租: 适合长期合作、固定课程

定价关系：
日租 ≈ 时租 × 6-8小时（优惠）
月租 ≈ 日租 × 20-25天（优惠）

示例：半亩天光讲堂
- 时租: RM50/小时
- 日租: RM350/天 (相当于7小时)
- 月租: RM8000/月 (相当于22-23天)
```

### 2. 空间设施配置

**必备设施：**
- ✅ 投影仪 - 讲座、培训
- ✅ 白板 - 教学、讨论
- ✅ 音响 - 大型活动
- ✅ 空调 - 舒适环境
- ✅ WIFI - 现代需求

**可选设施：**
- 讲台、舞台
- 视频会议设备
- 书架、阅读灯
- 茶水间

### 3. 活动与空间关联

**建议流程：**
```
1. 先规划活动
2. 根据活动规模选择合适空间
3. 在活动管理中选择对应空间
4. 日历自动显示活动和场地

好处：
✓ 避免场地冲突
✓ 资源利用可视化
✓ 收入来源清晰
```

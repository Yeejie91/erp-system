### 3️⃣ 自定义发票号码 🔢

**位置：** 新建发票表单最上方

**功能：**

**默认（系统生成）：**
```
┌────────────────────────────────┐
│ 发票编号          ☐ 自定义编号 │
│ ┌────────────────────────────┐ │
│ │ INV202410154567 (系统自动) │ │  ← 灰色只读
│ └────────────────────────────┘ │
└────────────────────────────────┘
```

**启用自定义：**
```
┌────────────────────────────────┐
│ 发票编号          ☑ 自定义编号 │
│ ┌────────────────────────────┐ │
│ │ [输入框可编辑]              │ │
│ │ 例如：INV-2024-001         │ │  ← 白色可输入
│ └────────────────────────────┘ │
└────────────────────────────────┘
```

**自定义格式示例：**
- `INV-2024-001`（年份流水号）
- `SWC-001`（公司缩写+流水号）
- `202410-001`（年月+流水号）
- 任意格式都可以！

- 中文在后（本地化）

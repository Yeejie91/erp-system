# 🏷️ 价格标签打印功能使用指南

## 功能概述

价格标签打印功能可以帮您快速打印商品价格标签，适用于：
- 📚 书店（图书标签）
- 🏪 零售店（商品标签）
- 📦 仓库（库存标签）
- 🎨 展会（展品标签）

## 标签包含信息

每个价格标签自动包含：
- ✅ **商品名称**（书名）
- ✅ **分类**（商品类别）
- ✅ **条形码** (Barcode)
- ✅ **入库年份**（从创建时间自动获取）
- ✅ **售价**（RM格式，红色醒目显示）
- ✅ **SKU编号**（小字显示）

## 使用步骤

### 第一步：在商品中添加条形码

1. 进入 **"商品管理"**
2. 添加或编辑商品
3. 填写 **"条形码 (Barcode)"** 字段
   - 可以是ISBN（图书）
   - 可以是EAN/UPC
   - 可以是自定义编号
4. 保存商品

### 第二步：选择要打印的商品

1. 进入 **"库存管理"**
2. 在商品列表中，每行前面有复选框 ☑️
3. 勾选需要打印标签的商品
   - 可以单选
   - 可以多选
   - 可以全选（点击表头的复选框）

### 第三步：打印标签

1. 选中商品后，页面上方出现蓝色提示框
2. 显示："已选择 X 个商品"
3. 点击 **"打印价格标签"** 按钮
4. 浏览器自动打开打印预览
5. 选择打印机并打印 🖨️

## 标签尺寸

**标准尺寸：60mm × 40mm**
- 适合大多数标签打印机
- 可用于标签纸打印
- 也可用A4纸打印（多个标签）

## 批量打印

### 方法1：选择多个商品

```
1. 勾选商品A ✓
2. 勾选商品B ✓
3. 勾选商品C ✓
4. 点击"打印价格标签"
5. 系统生成3个标签，每个标签独立分页
```

### 方法2：全选打印

```
1. 点击表头的复选框（全选）
2. 所有商品被选中
3. 点击"打印价格标签"
4. 批量打印所有商品标签
```

### 方法3：搜索后打印

```
1. 搜索"电子产品"
2. 筛选出所有电子产品
3. 点击全选
4. 批量打印所有电子产品标签
```

## 打印技巧

### 使用标签打印机

1. 设置打印机纸张大小为 **60mm × 40mm**
2. 选择打印机
3. 打印设置：
   - 边距：无边距
   - 缩放：100%
4. 打印

### 使用普通打印机（A4纸）

1. 购买A4标签贴纸（规格：60mm × 40mm）
2. 打印前预览
3. 调整打印设置
4. 打印后裁剪或撕下标签

### 打印前检查

- ✅ 商品名称是否正确
- ✅ 价格是否最新
- ✅ 条形码是否填写（如需要）
- ✅ 分类是否正确

## 常见问题

**Q: 可以修改标签尺寸吗？**
A: 可以。标签默认60mm×40mm，如需修改，可以联系技术人员调整CSS中的尺寸设置。

**Q: 条形码字段必填吗？**
A: 不必填。如果不填，标签上就不显示条形码部分，更多空间显示其他信息。

**Q: 入库年份从哪里来？**
A: 自动从商品创建时间（createdAt）提取年份，无需手动填写。

**Q: 可以打印多份同一个标签吗？**
A: 当前版本打印一次每个商品一张。如需多份，可以重复勾选打印，或使用打印机的"份数"设置。

**Q: 能打印中英文双语吗？**
A: 标签已包含部分英文标注（如"RM"），如需更多双语内容，可定制。

**Q: 如何取消选择？**
A: 点击蓝色提示框中的"取消选择"按钮，或逐个取消勾选商品。

## 最佳实践

### 图书标签流程

```
1. 商品管理 → 添加图书
   - 名称：书名
   - 分类：文学/科技/儿童等
   - 条形码：填写ISBN号
   - 售价：定价

2. 进货管理 → 录入库存

3. 库存管理 → 选择图书 → 打印标签

4. 贴标签到书上
```

### 促销商品标签

```
1. 商品管理 → 调整促销价格
2. 库存管理 → 选择促销商品
3. 批量打印新标签
4. 更换旧标签
```

### 新到货标签

```
1. 进货管理 → 录入新商品
2. 库存管理 → 筛选最新商品
3. 全选 → 打印标签
4. 贴标签到货品
```

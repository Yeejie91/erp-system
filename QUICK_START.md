# ERP系统快速启动指南

## 🚀 启动系统

### 1. 安装依赖
```bash
cd erp-system
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问系统
打开浏览器访问: `http://localhost:5173`

## 📋 核心功能测试清单

### ✅ 商品预留功能测试
1. **创建商品预留**
   - 进入"商品预留"页面
   - 点击"新增预留"
   - 选择预留类型（客户/活动/项目/事件）
   - 选择关联对象
   - 选择商品和数量
   - 保存预留

2. **查看预留状态**
   - 在预留列表中查看状态
   - 可以编辑或取消预留

### ✅ 展览库存功能测试
1. **创建展览库存**
   - 进入"展览库存"页面
   - 点击"新增展览"
   - 选择事件类型
   - 填写事件信息
   - 选择商品和数量
   - 保存展览

2. **管理展览状态**
   - 查看展览状态
   - 更新展览信息

### ✅ 活动管理集成测试
1. **创建活动预留**
   - 进入"活动管理"页面
   - 创建或编辑活动
   - 点击"预留商品"按钮
   - 选择商品和数量
   - 保存预留

2. **查看活动预留**
   - 在活动卡片中查看预留商品
   - 查看预留状态

### ✅ 库存管理测试
1. **查看预留数量**
   - 进入"库存管理"页面
   - 查看"预留数量"列
   - 查看"可用库存"列

2. **查看库存历史**
   - 点击商品的"查看记录"按钮
   - 查看包含预留和展览的记录

### ✅ 报表分析测试
1. **查看预留分析**
   - 进入"数据分析报表"页面
   - 切换到"预留分析"标签页
   - 选择时间范围
   - 查看预留统计数据

### ✅ 权限控制测试
1. **测试不同角色权限**
   - 使用不同角色账户登录
   - 测试各功能的访问权限
   - 验证权限控制是否生效

## 🔧 故障排除

### 常见问题
1. **页面空白**
   - 检查浏览器控制台错误
   - 刷新页面重试

2. **数据不显示**
   - 检查数据库连接
   - 重新加载数据

3. **权限错误**
   - 确认用户角色
   - 检查权限配置

### 重置系统
如果遇到问题，可以：
1. 清除浏览器缓存
2. 重新启动开发服务器
3. 重新登录系统

# 🏢 完整ERP系统功能清单

## 🎯 系统概览
**访问地址：** `http://localhost:3001`  
**系统名称：** 企业ERP管理系统  
**技术栈：** React + TypeScript + IndexedDB + Tailwind CSS

---

## 📊 核心功能模块

### 1. 🏠 仪表盘 (Dashboard)
- **统计卡片**
  - 商品总数和库存价值
  - 低库存预警
  - 本月利润和收入
  - 进行中活动数量
- **库存预警**
  - 自动检测库存不足商品
  - 显示当前库存和预警值
- **最近库存变动**
  - 显示最近5条库存变动记录
  - 包含入库/出库类型和数量
- **活动日程**
  - 显示即将开始的活动
  - 活动状态和时间信息
- **快速操作**
  - 添加商品、进货登记、出货登记、创建活动
- **业务日历**
  - 集成日历视图显示活动和收入

## 📋 使用建议

### 首次使用
1. 访问 `http://localhost:3001`
2. 使用默认账号登录（admin/admin123）
3. 点击"加载示例数据"了解系统功能
4. 根据需要配置系统设置

### 日常使用
1. 从仪表盘开始，查看整体运营状况
2. 使用快速操作进行常用操作
3. 定期查看报表了解业务趋势
4. 及时处理库存预警和应收账款

### 数据管理
1. 定期进行数据备份
2. 设置自动备份功能
